Trinket.io has ready-made, kid-targeted Pygame Zero courses, runs entirely
in-browser, zero install — a solid fallback if Pydroid 3's native-app
route proves unreliable on this specific tablet (especially if physical
keyboard passthrough turns out flaky). Weaker match to the "real dev
environment" goal since it runs on Skulpt (a JS reimplementation of
Python), not real CPython/pygame — and full pygame/GUI access may be
paid-tier only, unverified as of this writing.
