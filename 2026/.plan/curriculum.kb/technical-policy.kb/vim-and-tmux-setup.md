vim in one tmux pane, a shell pane beside it, mouse enabled in both
(`set mouse=a`, tmux mouse mode). Adult config pass before session 0:
syntax highlighting, a big font, and `set number` — line numbers are
load-bearing for traceback literacy.

Abby's complete vim vocabulary: `i` (start typing), `Esc` (stop), `u`
(undo), `:w` (save), click-to-move. **Fire drill:** editor acting weird?
`Esc`, then `u` until it looks right — always her hands; an editor rescue
the adult performs violates
`../design/020-goals.kb/abby-authors-the-code.md`. **Run ritual:** `:w`,
click the run pane, ↑, Enter.
