How Scratch concepts map to Python/pyxel, for a Scratch-fluent learner.
`direct-translations.md` and `genuinely-new-concepts.md` are compact
reference tables/lists. `misconceptions.kb/` holds the cases where Scratch
knowledge actively *misleads* — each is substantial enough (a specific
session tie-in, a "name it before she trips" instruction) to warrant its own
file. Add a misconception file when a new "guaranteed wrong prediction" is
identified; add to the other two files for anything shorter.
