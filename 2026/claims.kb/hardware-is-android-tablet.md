Abby's dedicated hardware is a Lenovo Idea Tab Plus / Tab K12 (ZAG70921US)
— an Android 15 tablet (Dimensity 6400, 8GB RAM, 12.1" touchscreen), not a
laptop, with a stand, external keyboard, and trackpad already available.
Plenty of horsepower for Python + pygame; the open question is software
compatibility (Android vs. desktop Python tooling), not performance.
