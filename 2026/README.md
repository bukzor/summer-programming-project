# Abby's Craft

A Minecraft-like game by Abby (age 10), in Python +
[pyxel](https://github.com/kitao/pyxel).

## Run it

```
cd projects/abbys-craft
uv run python main.py
```

`main.py` doesn't exist yet — it gets born in session 0, at Abby's
keyboard. After that, it grows a little every session. New games get
their own folder in `projects/`, always with a `main.py`, always run
the same way.

Game ideas go in `ideas.md` so they don't get lost. Experiments go in
`playground/` — nothing there can break the real game.
